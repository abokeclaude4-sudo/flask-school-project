# Flask School Project

## 📌 Overview

This is a Flask-based web application developed as part of a school project. It demonstrates backend development, routing, and basic web application functionality using Python and Flask.

## 🚀 Features

* User login system
* Dashboard functionality
* Password change feature
* Backend routing with Flask
* Basic HTML templates

## 🛠️ Technologies Used

* Python
* Flask
* HTML
* CSS

## 📂 Project Structure

* app.py → Main Flask application
* templates/ → HTML pages
* static/ → CSS and assets

## ▶️ How to Run

1. Install Flask:
   pip install flask

2. Run the application:
   python app.py

3. Open in browser:
   http://127.0.0.1:5000/
